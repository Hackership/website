[workshop] Intro to Functional Programming

Always wanted to learn more about functional programming, but not sure where to start? This is the workshop for you!

Functional programming solves a number of problems that imperative languages are traditionally less equipped to deal with. Like parallelization and predictability.

As a result, functional languages are gaining popularity and non-functional languages (like Python and JavaScript) have started to incorporate functional concepts. High time you learn how to leverage those!

During this afternoon we will introduce you to the basic concepts, using a language you probably already know and love: Python*. Together we will transform your code from a stateful mess to a functional masterpiece (just kidding... but really..).

April 25, 1-4pm SAN FRANCISCO
